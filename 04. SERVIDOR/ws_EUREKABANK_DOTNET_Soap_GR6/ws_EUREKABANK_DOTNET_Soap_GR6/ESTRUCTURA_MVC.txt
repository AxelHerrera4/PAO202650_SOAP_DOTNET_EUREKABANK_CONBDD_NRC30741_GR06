## 🔄 FLUJO DE UNA TRANSACCIÓN DE DEPÓSITO

```
1. CLIENTE SOAP
   └─ Depositar(cuentaId="001", monto=1000, empleadoId=1)

2. CONTROLADOR
   └─ EurekaBankWSControlador.Depositar()
      └─ Llama TransaccionService.RealizarDeposito()

3. SERVICIOS
   └─ TransaccionService.RealizarDeposito()
      ├─ GestorConcurrencia.BloquearCuenta()  ← Bloquea acceso
      ├─ Abre transacción SQL
      ├─ TransaccionService obtiene cuenta
      ├─ Calcula nuevo saldo
      ├─ Actualiza saldo en BD
      ├─ Registra movimiento
      ├─ Commit transacción
      └─ GestorConcurrencia.LiberarCuenta()  ← Libera acceso

4. DAO
   ├─ CuentaDAOImpl.ObtenerCuentaForUpdate()
   │  └─ SELECT * FROM cuenta WITH (UPDLOCK, ROWLOCK)
   │
   ├─ CuentaDAOImpl.ActualizarSaldo()
   │  └─ UPDATE cuenta SET saldo = @saldo
   │
   └─ MovimientoDAOImpl.RegistrarMovimiento()
      └─ INSERT INTO movimiento (...)

5. BASE DE DATOS
   └─ Operaciones confirmadas ✅

6. RESPUESTA AL CLIENTE
   └─ SUCCESS: Depósito realizado. Nro Operación: DEP-1234567890
```
